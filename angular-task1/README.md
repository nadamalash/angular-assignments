## Getting Started

### Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/nadamalash/angular-task1.git
   ```

2. Navigate to the project folder:

   ```bash
   cd angular-task1
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the app:

   ```bash
   ng serve
   ```

5. Open your browser at http://localhost:4200.
